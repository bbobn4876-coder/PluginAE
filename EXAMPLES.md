## Базовая цветокоррекция

### Пример 1: Теплая атмосфера заката

**Цель**: Придать видео теплый, закатный тон

**Шаги**:
1. Примените PluginAE к слою
2. Выберите пресет: **Warm Sunset**
3. Установите Intensity: **65%**
4. При необходимости добавьте второй слой PluginAE:
   - Пресет: **Vignette**
   - Intensity: **35%**

**Результат**: Теплое, уютное изображение с оранжево-красными тонами

**Применение**: Романтические сцены, закатные кадры, уютные интерьеры

---

### Пример 2: Холодный кинематографический стиль

**Цель**: Создать современный холодный "голливудский" вид

**Настройки**:
```
Слой 1:
- Preset: Cinematic Blue
- Intensity: 70%

Слой 2 (опционально):
- Preset: Teal & Orange
- Intensity: 40%
- Blend Mode: Screen
```

**Результат**: Профессиональный кинематографический вид с холодными синими тонами в тенях

**Применение**: Боевики, триллеры, научная фантастика

---

### Пример 3: Драматичный черно-белый

**Настройки**:
```
Слой 1:
- Preset: Black & White
- Intensity: 100%

Слой 2:
- Preset: Vignette
- Intensity: 55%

Слой 3:
- Preset: Film Grain
- Intensity: 25%
```

**Результат**: Контрастное черно-белое изображение с виньетированием и зерном

**Применение**: Артхаус, документалистика, драматические моменты

---

## Кинематографические эффекты

### Пример 4: Голливудский Teal & Orange

**Один из самых популярных стилей в современном кино**

**Рецепт**:
```
1. PluginAE:
   - Preset: Teal & Orange
   - Intensity: 75%

2. Встроенный Curves (опционально):
   - Усильте контраст в средних тонах

3. PluginAE (второй экземпляр):
   - Preset: Vignette
   - Intensity: 40%
```

**Фильмы с этим стилем**: Mad Max, Transformers, The Matrix

---

### Пример 5: Винтажный кинофильм

**Создание эффекта старого кино 70-80х**

**Полная настройка**:
```
1. PluginAE - Vintage Film:
   - Intensity: 85%

2. PluginAE - Film Grain:
   - Intensity: 35%

3. PluginAE - Vignette:
   - Intensity: 50%

4. Встроенный Glow (AE):
   - Threshold: 80%
   - Radius: 50
   - Intensity: 0.3

5. Встроенный Sharpen (AE):
   - Amount: -20 (легкое размытие)
```

**Результат**: Теплые тона, зерно, мягкое изображение

## Переходы между сценами

### Пример 6: Плавное затухание

**Создание fade-to-black перехода**

**Анимация**:
```
Кадр 0:00:00 - Intensity: 0%
Кадр 0:00:15 - Intensity: 100%
Кадр 0:01:00 - Intensity: 100%
Кадр 0:01:15 - Intensity: 0%
```

**Настройки**:
- Preset: Fade In/Out
- Easy Ease на всех keyframes

---

### Пример 7: Динамический переход с zoom blur

**Энергичный переход между сценами**

**Настройки**:
```
Сцена 1 (последние 10 кадров):
- Preset: Zoom Blur
- Intensity: 0% → 100% (анимация)

Сцена 2 (первые 10 кадров):
- Preset: Zoom Blur
- Intensity: 100% → 0% (анимация)
```

**Применение**: Музыкальные клипы, спортивные видео, динамичные сцены

## Стилизация под винтаж

### Пример 8: Instagram-стиль "1977"

**Имитация популярного Instagram фильтра**

```
1. PluginAE:
   - Preset: Warm Sunset
   - Intensity: 45%

2. Встроенный Tint:
   - Map Black to: RGB(30, 20, 15)
   - Map White to: RGB(255, 245, 235)
   - Amount: 30%

3. PluginAE:
   - Preset: Film Grain
   - Intensity: 18%

4. Встроенный Curves:
   - Поднимите черные до RGB(20, 20, 20)
```

---

## Продвинутые техники

### Пример 13: Выборочная цветокоррекция

**Применение эффекта только к определенной области**

**Метод**:
1. Создайте маску на слое с PluginAE
2. Настройте Feather для плавного перехода
3. Примените эффект внутри маски

**Применение**: Цветокоррекция лица, выделение объекта

---

### Пример 14: Анимированная интенсивность по музыке

**Синхронизация эффекта с битами**

**Шаги**:
1. Примените PluginAE
2. Alt+Click на Intensity
3. Добавьте Expression:
```javascript
audioLayer = thisComp.layer("Your Audio");
audioLayer.audio.audioLevels[0] * 100
```

**Результат**: Intensity пульсирует в такт музыке

---

### Пример 15: Градиентный переход эффектов

**Плавный переход от одного пресета к другому**

**Техника**:
1. Дублируйте слой
2. Слой 1: Preset A, Opacity анимация 100% → 0%
3. Слой 2: Preset B, Opacity анимация 0% → 100%
4. Синхронизируйте анимации

**Применение**: Смена времени суток, смена настроения

## Шаблоны для быстрого доступа

### Сохранение как Animation Preset

1. Настройте все слои PluginAE
2. Выберите все эффекты
3. **Animation → Save Animation Preset**
4. Назовите, например: "My Cinematic Look"
5. Применяйте одним кликом в будущем!

## Рекомендации

### Когда использовать низкую интенсивность (10-30%)
- Тонкая цветокоррекция
- Корпоративное видео
- Документальное кино

### Когда использовать среднюю интенсивность (40-70%)
- Обычная цветокоррекция
- Большинство проектов
- Музыкальные клипы

### Когда использовать высокую интенсивность (80-100%)
- Стилизация
- Креативные проекты
- Специальные эффекты
