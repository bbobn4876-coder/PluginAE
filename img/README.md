# Icons

This folder contains icon assets for the AEP Preview plugin.

## Required Icons

For proper display in Adobe After Effects, replace the placeholder SVG files with actual PNG images:

- `icon-dark.png` - Icon for dark theme (32x32 pixels recommended)
- `icon-light.png` - Icon for light theme (32x32 pixels recommended)

## Icon Guidelines

- **Format**: PNG with transparency
- **Size**: 32x32 pixels (minimum), can provide @2x versions at 64x64
- **Style**: Simple, monochromatic design that works well at small sizes
- **Theme**: Dark version should have light-colored graphics, light version should have dark-colored graphics

## Creating Icons

You can use any image editor like:
- Adobe Photoshop
- Adobe Illustrator (export as PNG)
- Figma
- Sketch
- Free tools like GIMP or Inkscape
